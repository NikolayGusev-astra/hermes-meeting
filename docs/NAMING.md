# Соглашение об именах и структуре выходных артефактов


## Правила именования папок

Формат: `{YYYY-MM-DD}_{тип}_{тема}`

- **Дата** — из метаданных файла или `дата_не_установлена`
- **Тип** — `встреча`, `лекция`, `интервью`, `презентация`
- **Тема** — 2-5 ключевых слов из первых 30 секунд транскрипта, транслитом или кириллицей через дефис

Примеры:
- `2026-04-09_встреча_KPI-ИИ-спорт`
- `2026-07-22_лекция_Arista-Pro-разграничение-прав`
- `дата_не_установлена_встреча_замечания-методики`

## Правила для агента

1. После транскрибации — прочитать первые 30 секунд, определить тему.
2. Сформировать имя папки по шаблону.
3. Все файлы — только русские имена.
4. Если дата неизвестна — `дата_не_установлена`.
5. Дубликаты тем разрешать суффиксом `_2`, `_3`.
